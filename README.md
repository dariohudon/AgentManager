# Agent Manager

Native macOS app for managing reusable AI agents, prompts, workflows, handoffs, and review packets.

Agent Manager replaces the old Prompt Manager Chrome extension concept with a clean local-first macOS/Xcode project.

## Working Model

- ChatGPT Web = Architect
- Claude Code = Implementer
- Codex = Reviewer
- BATON = workflow/project truth

## V1 Goal

Create a local Agent Manager app where Dario can define agents, store prompts, organize workflows, copy/export agent instructions, and preserve handoff/review patterns.
